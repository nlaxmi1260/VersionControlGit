  4. Basic Git Commands
    1. git init : Initialize a new Git repository in the current directory.
    2. git clone < repository url > : Clone a remote repository to your local machine.
    3. git status :Show the status of the working directory and staging area, displaying changes and untracked files.
    4. git add < file-name > :Add files to the staging area. This command prepares your files for a commit.
    5. git commit -m "commit message" : Commit the staged changes to the local repository with a message.
    6. git configure local to remote git config --global --edit
    7. git log : Show the commit history for the current branch.
    8. git diff : Show changes between commits, branches, or the working directory and the index.
    9. git push origin <branch-name > : Push your local commits to a remote repository.
    10. git pull origin < branch-name >: Fetch changes from a remote repository and merge them into the current branch
    11. git checkout < branch-name > : Switch branches or restore working tree files.
    12. git branch :
      1. git branch : To list branches
      2. git branch < branch-name >: to create a branch
      3. git branch -d < branch-name > : to delete a bramch
    13. file status : untracked -> unstage -> modified
