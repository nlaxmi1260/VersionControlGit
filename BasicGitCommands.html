Git Commands

Basic Git Commands

  1. git init: Initialize a new Git repository in the current directory
  2. git status: Shows the status of working directory and staging area, displaying changes and untracked files.
  3. Untracked files are the files which git is not tracking.
  4. git add < file-name >: Add given file to the stageing area.
  5. git add . : Add all untracked files to the stageing area.
  6. git rm --cached < file-name > : Remove file from staging area to untracked area.
  7. git commit -m "your coomit message": Commit the staged changes to the local repository with a message.
  8. git log: Show the commit history for the current branch.
  9. git diff: Show changes between commits ,branches or the working directoy and the index.
  10. git help commit:Help for the commit command.
  11. git checkout < branch-name >:Switch branches or restore working tree files.
  12. git branch:To list branches
  13. git branch < branch-name >:To create a new branch.
  14. git branch -d < branch-name >:To delete a branch.
  15. git rm < file_name >:To remove a file.
  16. git restore < file_name >to discard changes in working directory i.e previous commit changes will be avalable.
  17. git restore --staged < file_name >
  18. git rm < file_name > : To remove a file
  19. git rm --cached < file_name > : staged to untracked
  20. Loving Git !!!
